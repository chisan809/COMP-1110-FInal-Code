# Complete Budget App Testing Package

This package contains four self-contained testing cases.

## How to run
1. Choose one case folder.
2. Open Terminal.
3. `cd` into that folder.
4. Run `python3 budget.py`.
5. Follow the README inside that case folder.

## Cases
- Case 1: HKU student stress test
- Case 2: Group payment and reimbursement
- Case 3: Subscription creep and recurring transactions
- Case 4: Travel preparation and rollover budgeting

Each folder includes the program, CSV files, budget files, and instructions.
